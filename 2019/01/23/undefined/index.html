我先以这个网址上的所有内容作为一个学习的对象。以下所有内容都是我自己的一些笔记。如果有什么含糊不清的地方。可以点击上方的传送门进行查漏补缺。有问题请指出。我会在以后慢慢给自己的网站部署留言的地方。（虽然感觉就是复制粘贴人家的代码的事。但是现在还是好懒呀）

js与java关系甚少，但是基础语法很像（哪个语言语法不像……）作者借java的名头，让js更火，此为js的由来。

js的用途

用于在静态网页上增加动态效果

js语法

以下内容会很繁杂，不喜勿喷


  • js可以镶嵌在网站的各个地方。具体是

    < script > ………………< /script>

    //在进行上述操作的时候，可能是布局在md上的原因，有一些冲突。具体可以写md的时候自己试一下。

    不过通常规定是放在head中

    以下

    < head>

    > < script >…… </ script>

    < /head>

    其中的js代码它可以直接被浏览器执行

    • 第二种方法，将js代码放入到一个单独的.js文件中，更有利于去维护代码，并且多个页面可以引用同一份.js文件。

    具体实现方法

    < head>

    ​ < script src=”/xx/xx/x.js” >< /script>

    < /head>

    type属性有时候会出现，但是没必要。

名字空间

全局变量会绑定到window上，不同的js文件如果使用了相同的全局变量，或者定义了相同名字的顶层函数，都会造成命名冲突，并且难以被发现。

减少冲突的方法就是设定一个对象，把所有的变量和函数都绑定这个全局对象中。

局部作用域

js 的变量的作用域实际上是在函数内部。在for循环等语句块中是无法定义具有局部作用域的变量。//既在for循环语句之外，还能用在for循环里面定义的函数变量。符合了上一章所说的。在一个函数开始前，扫描函数所有的语句，将定义的语句提到函数的顶部。

引入let，可以局部定义一个块级作用域的变量。

常量

使用 const来定义常量，且const和let都具有块级作用域。

解构赋值

解构赋值，可以同时对一组变量进行赋值。

var [x,y,z]=[‘hello’,’javascript’,’es6’];

注意

对数组元素进行解构赋值的时候，多个变量要用[……]括起来。

如果数组本身还有嵌套，也可以通过下面的形式的解构赋值，注意嵌套层次和位置要保持一致。

let [x,[y,z]]=[‘hello’,[‘javascript’,’es6’]];

解构赋值还可以忽略某些元素。

从对象中取出若干属性，也可以使用解构赋值。

var person = {
name: ‘小明’,
age: 20,
gender: ‘male’,
passport: ‘G-12345678’,
school: ‘No.4 middle school’
};
var {name, age, passport} = person;

同样可以直接对嵌套的对象进行赋值，保证对应的层次是一致的。

方法

在一个对象中绑定函数，称为这个对象的方法。

this

this是一个特殊变量，它始终指向当前对象。this.xx既拿到当前对象的xx属性值。

如果在函数内定义函数，this指向undefined，在对象外定义函数也是这样。

apply

this是可以控制指向的。调用函数本身的apply的方法，它接收两个参数，第一个参数就是绑定this，第二个参数是数组，表示函数本身的参数。

利用apply修复函数的调用。

call（）

与apply()类似的方法是call(),唯一的区别是：

·apply把参数打包成数组传入。

·call把参数按顺序传入。

装饰器

利用apply，我们可以动态改变函数的行为。

js 的所有对象都是动态的。既是内置函数，我们也可以重新指向新的函数。

闭包

返回函数不要引用任何循环变量，或者后续会发生变化的变量。

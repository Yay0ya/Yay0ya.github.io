提醒

变量提升

js的函数有个特点，它会先扫描函数体的所有语句，把所有声明的变量提升到函数的顶部。他提升了变量的声明（声明如果没有东西，默认为undefined），但是不进行遍历的赋值。

为了防止出现以上情况。完成所有的变量声明在函数的顶部。

Number

Number作为一个数据类型，满足所有的四则运算行为

字符串

用单引号\’或者双引号\”括起来的文本。具体参考c++。

布尔值

ture or false ;与c++相似

&&与运算

比较运算符

更多的判别在

==

以及

===

==比较，它会自动转换数据类型再进行比较（考虑字符串和数字的比较）

===比较，不会自动转换数据类型。数据类型不一致，返回false。一致后，再进行==比较。

特殊

NaN这个特殊的Number值与其它的值都不相等。包括它自己。intersting

唯一能判断NaN的方法是通过isNaN()函数：

isNaN(NaN);//true

最后注意浮点数的相等比较

浮点数再运算过程中会产生误差。计算机无法精确循环小数。感觉又和c 的double一样。然后观察器是否小于某个阈值。

Math.abs(1/3-(1-2/3))<0.00000001;//true

null和undfined

null表示一个空的值，它和0以及空字符串不同。0是一个数值，\’ \’ 长度为0的字符串，而null表示空。

undefined 表示未定义。用于判断参数是否传递的情况的下有用。

数组

按顺序排列的集合，集合的每个值称为元素。js中数组可以包括任意的数据类型。

例如：

[1,2,3.14,\’hello\’,null,true]

数组用[]表示，元素之间用,分隔。

另一种创建数组的方法是通过Array()函数实现：

new Array(1,2,3)；//创建了数组[1,2,3]

出于可读性的考虑，建议直接使用[]。

数组的元素可以通过索引来访问。索引的初始值为0。

与c相似，索引超出了范围则返回undefined。

对象

js的对象是一组由键-值组成的无序集合，例如：

var person={

​ name:\’Bob\’,

​ age:20,

​ ……

}；

js对象的键都是字符串类型，值可以是任意数据类型。

定义的键值对：name:\’Bob\’，每个键又称为对象的属性

person的name属性为\’Bob\’

获取一个对象的属性，我们用 对象变量.属性名的方式。（然后为了加粗研究了一下md文档的标识方式。ctrl+B完成一个段落内的文字加粗的方式。然后####的方式只能在一个段落的开头标识，表示这是一个标题)。

变量

变量名的命名与以前学过的c和c++差不多。这里安利一个变量命名的网站。

命名网站这个网站是github上的一个项目。网页制作很精良。（大概我想我本次科创的项目设计的网站样式仿造这个）

声明一个变量用var语句。

如：

var a;

var b=1；

var c=\’133434\’

var d=true;

var t=null;

变量名也可以是中文名，但是有点给自己找麻烦的嫌疑。（反正c和c++肯定不认中文变量名）

=进行复制。

同一个变量可以反复赋值。而且可以是不同类型的变量。但是注意只能用var声明一次。（变量本身类型不固定的语言称之为动态语言）

如果一个变量没有通过var声明就进行使用，那么该变量就自动被声明为全局变量。（有点像python中间的语法规定）

为了防止冲突，引入了strict模式。（该模式强制要求所有变量需要声明，如果没有声明则进行报错。）

启用strict模式的方式。在js的第一行写上：

\’use strict \’

这是一个字符串，不支持strict模式的浏览器会把它当作一个字符串语句执行。

对字符进行一些操作，类似字符数组的索引。

或者取字符串的长度，不包括\0。

toLowerCase

substring(0,5)；//从第一个字符串开始返回，不包括第五个字符

substring(7);//从第七个字符串开始返回至末尾

数组

上面大致讲诉了数组的作用。数组可以包含任意的数据类型。

取得一个数组的长度可以使用数组的length属性。

不同于字符串，如果给数组的length赋一个新值，会将数组的大小进行改变。

改变数组的索引的字也可以改变数组中的值。

indexOf

搜索元素的位置

slice

返回一个区间值。

slice(0,3)类似于字符串里面的substring();

如果不传递任何参数，它就可以从头到尾的截取所有的元素。利用这个特点，可以复制一个数组。

push和pop

push()向数组的末尾添加若干元素。

pop()抹除最后一个元素

push(a,b);

pop();

unshift和shift

向一个数组的的头部添加或者删除元素。

unshift()向数组头部增加若干元素。

shift()把数组的第一个元素删除。

sort

sort()对当前数组进行排序，可以直接修改当前数组的数组位置，直接调用的时候按照默认顺序排序。也可以根据指定要求进行顺序排序。

reverse

按照当前顺序，逆序存放所有数据。

splice

修改数组的一个方法，可以做到从指定的索引开始删除若干元素，然后可以再从该位置进行添加。

arr.splice(2，3，\’google\’,\’facebook\’);

concat

将两个数组相联。并返回一个新的数组。

没有修改当前的数组，而是返回一个新的数组。concat方法可以接收任意个元素和数组，并且自动把所有的数组拆开，添加到新的数组里面。

join

把数组中的每个元素用特定的字符串连接起来，返回连接后的字符串。

如果元素不是字符串，则自动转换为字符串后再连接。

Map和Set

js中对象的键必须是字符串。

Map

Map是一组键值对的结构，查找速度快。

var m=new Map([[\’michael\’,95],[\’Bob\’,75],[\’Tracy\’,85]]);

m.get(\’micheal\’);//95

初始化map需要一个二维数组，或者直接初始化一个空Map。

Map的方法：

m.set(\’Adam\’,67);//增加新的键值对

m.has(\’Adam\’);//判断是否存在键Adam

m.get(\’Adam\’);//取出Adam中的值

m.delete(\’Adam\’)；//删除键adam

键值对 key-value

Set

set与map相似。也是一组key组合，但不存储值。且key不重复，所以再set中没有重复的key

要创建一个Set，需要提供一个数组作为输入，或者直接创建一个空的Set

var s1=new Set();//空Set

var s2=new Set([1,2,3]);//含1，2，3

通过add可以完成添加元素。

delete完成删除元素。

literable

遍历数组可以采用下标循环，遍历Map或Set无法采用下表循环。

引入lterable类型集合可以通过for……of循环来遍历。

‘use strict’;
var a = [1, 2, 3];
for (var x of a) {
console.log(x);
}

区别 for……of及for……in循环

for……in循环由于一些问题，它遍历的实际上是对象的属性名称。

一个数组实际上也是一个对象，它的每个元素的索引被视为一个属性。

具体我好像有些难以理解。咕，但是大概知道是什么个意思。

函数

在js中的函数定义方式如下：

​ function abs(x){

……

}

·function指出这是一个函数定义

·abs是函数的名称

·(x)括号内列出函数的参数。

·{……}代码是函数体

与c相似，遇到return 函数执行完毕，并返回函数值。

如果没有return 语句。函数执行完毕后也返回结果，结果为undefined。

js的函数也是一个对象。

可以通过下面这种方式定义函数

var abs=function(x){

……

}；

这种方式下function(x)成为一个匿名函数。它没有函数名，但是这个函数赋值给了abs，所以可以通过变量abs调用该函数

两种方式等价。但是要在函数体末尾加一个;来表示赋值语句的结束。

调用函数

传入的参数值可以比规定的参数值多。且不会报错，也没有影响。

参入比规定的少也没关系。默认未传入的量就将参数认为是undefined，计算的结果会是NaN

argument

argument，它只在函数内部起作用，并且永远指向当前函数的调用者传入的所有参数。

argument类似一个数组，却不是数组。

利用argument可以获得调用者传入的所有参数。

可以利用argument.length来判断传入参数的个数。

rest

用于放在函数的参数接收的最末尾。以用来方便所有的多余的参数的判定。具体移步传送门。
